### Data Processing
- **Observation/Action:** Filters training samples using `SN_filter == 1`, retaining only sequences where the mean signal-to-noise ratio across all experimental conditions exceeds 1.0.
- **ML Rationale (Why it works):** Experimental biochemical measurements contain heteroscedastic noise. Low S/N samples have target values dominated by measurement error rather than true biological signal. Training on these introduces high-variance gradients that obscure the underlying degradation function. By filtering, the model learns from a high-fidelity subset, reducing irreducible label noise and aligning the training distribution with the cleaner, curated private test set.
- **Guiding Principle:** In scientific or crowdsourced datasets, always quantify and filter by experimental confidence metrics (S/N ratio, replicate variance, error bounds). A smaller, high-signal dataset consistently outperforms a larger, noisy one in regression tasks because it lowers the Bayes error rate and stabilizes gradient updates.

### Feature Engineering
- **Observation/Action:** Encodes primary sequence, secondary structure (dot-bracket), and loop type as independent categorical embeddings. Sums these embeddings with a linearly projected `pair_mask` (binary indicator of base-pairing) and learnable positional embeddings.
- **ML Rationale (Why it works):** RNA stability is a multi-scale phenomenon: local nucleotide chemistry dictates immediate susceptibility, while 2D folding creates long-range stabilizing/destabilizing interactions. Separate embeddings preserve modality-specific semantics before fusion. The explicit `pair_mask` injects a strong structural inductive bias, allowing the model to instantly recognize which distant positions interact, bypassing the need for the network to infer pairing solely from sequence patterns. Additive fusion enables cross-modal modulation without forcing early non-linear entanglement.
- **Guiding Principle:** For structured sequential data (proteins, RNA, polymers), explicitly encode domain-specific structural priors (pairing, spatial proximity, functional motifs) as separate feature channels. Fuse them additively or multiplicatively before the main network to let the model learn cross-modal interactions efficiently, rather than relying on implicit pattern discovery.

### Model Architecture
- **Observation/Action:** Implements a hybrid CNN-Transformer: a shallow 1D CNN stem (2 layers, kernel=5) processes local context, followed by a 4-layer Transformer encoder for global sequence modeling. Uses gradient checkpointing and mixed precision.
- **ML Rationale (Why it works):** The CNN stem acts as a localized motif extractor and denoiser. A kernel size of 5 captures biologically relevant pentamer/hexamer contexts that heavily influence local degradation rates. By compressing and smoothing local features first, the subsequent self-attention mechanism operates on a cleaner, more abstract representation, drastically improving sample efficiency and reducing attention overfitting. The Transformer then captures long-range dependencies (e.g., how a base at position 5 interacts with position 60 via folding). Checkpointing/AMP enables training this architecture within GPU memory limits without sacrificing depth.
- **Guiding Principle:** Combine local receptive field models (CNNs, small RNNs) with global context models (Transformers, GNNs) for sequential data. The local front-end serves as a learnable feature compressor and noise filter, making the global attention mechanism more robust, data-efficient, and less prone to memorization on small datasets.

### Training & Validation
- **Observation/Action:** Uses 3-fold K-Fold CV with early stopping on validation MCRMSE. Applies target-specific loss weighting (`[1.5, 1.5, 1.0, 1.5, 1.0]`), gradient accumulation (steps=4), and cosine annealing. Monitors train-val loss gap for overfitting detection.
- **ML Rationale (Why it works):** Multi-output regression targets often differ in scale, variance, and learnability. Uniform MSE loss causes high-variance targets to dominate gradient updates, starving low-variance targets. Empirical loss weighting balances per-task gradient magnitudes, ensuring all outputs converge harmoniously. Gradient accumulation simulates a larger effective batch size, stabilizing BatchNorm statistics and AdamW momentum. K-Fold + strict early stopping on the exact evaluation metric (MCRMSE) prevents over-optimization to training noise while maximizing generalization to unseen sequences.
- **Guiding Principle:** Align the training objective with the evaluation metric and target statistics. Use loss weighting to balance multi-task regression, especially when targets have heterogeneous scales or noise levels. Combine K-Fold cross-validation with metric-driven early stopping and learning rate scheduling to extract maximum generalization performance from limited data.
